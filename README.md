Задание по проектному практикуму, роль разработчик
В системе реализованы две основные роли студент и работодатель. 
Студент может откликаться, просматривать вакансии, отслеживать статус заявки, а также вступать в чат с работодателем
Работадатель может создавать вакансии, менять их статус, отсматривать заявки и вести переписку со студентом
Ключевое требование: два языка. Я использовал английский и испанский языки. Локализация распространяется на интерфейс и на содержание заявок.
